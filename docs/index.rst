Summary
=======
Fwog is a low-level OpenGL 4.6 abstraction written in C++20 which aims to offer improve ergonomics and eliminate deprecated functionality.

Getting Started
===============
::

    git clone https://github.com/JuanDiegoMontoya/Fwog.git
    cd Fwog
    mkdir build
    cd build
    cmake ..
